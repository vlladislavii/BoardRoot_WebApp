# BoardRoot – Board Game Club & Rental Platform

### Project Description
**BoardRoot** is a web application for board game enthusiasts. Unlike a standard shop, this platform is built for a club experience. You can use it to:
* **Rent and Go:** Pick a game from our catalog to play at home.
* **Stay and Play:** Reserve a table to enjoy games directly at the club.

The app manages everything from user registration and secure login, as well as tracking how many copies of a game are available at any moment.

### User Stories (Informal)

* Users can register a new account and log in.
* Clients can browse the board game catalog online.
* Clients can rent a board game for a specific time period.
* Clients have the option to add a table reservation to their game rental to play directly at the club.
* Clients can download game rulebooks in PDF format from the web site.
* Clients can upload photos from their gaming sessions to their profile.
* Admins can add new board games to the system.
* Admins can update the number of available game copies and tables.
* Admins can change the status of a booking manually (e.g., to "Completed").
